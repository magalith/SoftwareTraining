错误代码

错误代码 错误类型 详细错误
1001 权限错误 该接口需要登陆才能访问
1002 权限错误 当前用户无权限访问指定内容
1003 权限错误 错误的用户权限
2001 请求错误 POST参数错误
2002 请求错误 错误的数据格式
2003 请求错误 附件超过指定大小
3001 会话错误 会话已过期,或遭到中间人攻击
3002 会话错误 会话异常,请重新登陆后尝试
4001 服务异常 数据库访问异常
4002 服务异常 服务器出现未知错误
4003 服务异常 错误的查询结果 
返回格式: 
{
    "code": "1001",
    "data": {
        "type": "权限错误",
        "detail": "该接口需要登陆才能访问"
    }
}

通用接口

  1. 查看自己的信息

  2. 查看所有可选项目

  • 请求方法: POST
  • 请求地址: /api/get_self_info
  • 请求参数:
{
    "session": <会话ID>,
    "timestamp": <整数时间戳>
}
  • 正确返回结果:
{
	"code": "ok",
	"data": {
        "id": "U_001",
        "name": "江泽民",
        "gender": "男",
        "class": "国机二班",
    }
}

管理员接口

  1. 管理员查看全部学生名单
  • 请求方法: POST
  • 请求地址: /api/get_students
  • 请求参数:
{
    'session': <会话ID>,
    'timestamp': <整数时间戳>
}
  • 正确返回结果:
{
	"code": "ok",
	"data": [{
		"id": 1,
		"stuname": "aa",
		"class": "CLASS_01",
		"teacher": "teacher_a",
		"note": "项目一"
	}, {
		"id": 2,
		"stuname": "bb",
		"class": "CLASS_02",
		"teacher": "teacher_a",
		"name": "项目二"
	},
    {...}]
}
  1. 管理员更新班级信息(全部POST)
  • 请求方法: POST
  • 请求地址: /api/set_classes
  • 请求参数:
{
    'session': <会话ID>,
    'timestamp': <整数时间戳>
    "classes": [{
		"class_name": "CLASS_01",
		"teacher": "teacher_a",
        "students": [1001, 1002, 1003, ...]
	},
    {...}]
}
  • 正确返回结果:
{
	"code": "ok",
    // 班级ID
	"data": "C_001"
}
  1. 管理员获取教师名单
  • 请求方法: POST
  • 请求地址: /api/get_teachers
  • 请求参数:
{
    'session': <会话ID>,
    'timestamp': <整数时间戳>
}
  • 正确返回结果:
{
	"code": "ok",
	"data": [{
		"id": 1,
		"name": "aa",
		"class": ["CLASS_01", "CLASS_02"],
	}, {
		"id": 2,
		"name": "bb",
		"class": ["CLASS_03"],
	},
    {...}]
}
  1. 管理员获取所有班级名单(带学生老师名单)
  • 请求方法: POST
  • 请求地址: /api/get_class_info
  • 请求参数:
{
    'session': <会话ID>,
    'timestamp': <整数时间戳>
}
  • 正确返回结果:

	"code": "ok",
	"data": [{
		"cid": "C_001",
		"name": "aa",
        "teacher_id": "T_001",
		"students_id": ["S_001", "S_002"],
	},
    {...}]
}

教师接口

  1. 查看负责班级的学生名单
  • 与管理员-获取全部学生接口相同
  1. 查看所有阶段
  • 请求方法: POST
  • 请求地址: /api/get_stage
  • 请求参数:
{
    'session': <会话ID>,
    'timestamp': <整数时间戳>
}
  • 正确返回结果:
{
	"code": "ok",
	"data": [{
		"id": 1,
		"name": "阶段一",
		"teacher_id": "T_001",
	}, {
		"id": 2,
		"name": "阶段二",
		"teacher_id": "T_001",
	},
    {...}]
}
  1. 为阶段添加任务
  • 请求方法: POST
  • 请求地址: /api/push_mission
  • 请求参数:
{
    {
    "session": <会话ID>,
    "timestamp": <整数时间戳>,
    "stage_id": <阶段ID>,
    "mission": {
        "text": "<阶段文本内容>",
        "file": "<附件Base64编码>"
    }
}
  • 正确返回结果:
{
	"code": "ok",
	"data": "<任务ID>"
}
  1. 查看某一任务的所有文档
  • 请求方法: POST
  • 请求地址: /api/check_mission
  • 请求参数:
{
    "session": <会话ID>,
    "timestamp": <整数时间戳>,
    "mission_id": <整数任务ID>
}
  • 正确返回结果:
{
	"code": "ok",
	"data": [
        {
            "id": "D_001",
            "text": "第一次作业.......",
            "file": "http://0.0.0.0/a.txt",
            "student_name": "aaaaaaa",
            "score": 85
        },{
            "id": "D_002",
            "text": "第二次作业.......",
            "file": "http://0.0.0.0/b.txt",
            "student_name": "bbbbbbb",
            "score": 90
        },
    ]
}
  1. 更新文档(学生的作业)分数
  • 请求方法: POST
  • 请求地址: /api/set_score
  • 请求参数:
{
    "session": <会话ID>,
    "timestamp": <整数时间戳>,
    "mission_id": <整数任务ID>,
    "score":[
        {
            "D_id": "D_001",
            "score": 95
        },{
            "D_id": "D_002",
            "score": 98
        },
    ]
}
  • 正确返回结果:
{
	"code": "ok",
	"data": "Updated"
}

  • 请求地址: /api/get_students_doc
  • 请求参数:
